## Welcome

RaceCoordinator AI is a revolutionary new take on the traditional RC Race Coordinator software.  It uses the latest in web technologies to provide a modern, user-friendly interface for managing your slot car races.  It is the newest version of the original [RaceCoordinator](https://www.racecoordinator.net) software used for the last 10+ years by many basement racers and slot car clubs around the world.

RaceCoordinator AI is an open source project found here:
[RaceCoordinator AI](https://github.com/daufderheide/racecoordinator_ai)
All contributions are welcome!
